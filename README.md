# spa-page-ractive

SPA project based on page.js and ractive.js

## Requirements

1. node.js
2. packages
  - `npm install -g gulp`
  - `npm install -g cult`
    (or `npm install -g coffee-script`, `coffee -c gulpfile.coffee`, and replace all `cult` with `gulp`)

## Building
1. `npm install`
2. `cult` to run a test server on localhost and rebuild on file changes
